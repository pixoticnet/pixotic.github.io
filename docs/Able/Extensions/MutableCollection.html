MutableCollection

public extension MutableCollection where Element : Identifiable

Available where Element : Identifiable

  • subscript(selecting:)

    Selects the elements of the given collection using a SelectionOf.

    Note

    this subscript assumes that ID will have at most a single representation in the collection in order to optimize the traversal and short-circuit once all the identifiers have been found.
